# App Spec: YouTube Downloader + Offline Player

## Type: full app

## Sites
### youtube.com
- Target URLs: https://www.youtube.com/@veaborea (Veritasium), https://www.youtube.com/@3blue1brown
- Transport: InnerTube POST API (youtubei/v1/*), no auth needed, API key from page
- Key endpoints:
  - POST /youtubei/v1/search — search channels/videos
  - POST /youtubei/v1/browse — channel page with video list
  - POST /youtubei/v1/player — video details, streaming formats, captions
  - GET /feeds/videos.xml?channel_id=X — RSS feed per channel
  - GET suggestqueries — JSONP autocomplete
- Auth: public, just needs API key + client context in POST body
- Pagination: continuation tokens in response → pass back in next request
- Fields needed: video title, thumbnail, duration, view count, upload date, description, available qualities, channel name/avatar

## Download
- yt-dlp via Python bridge for actual downloads
- Store in data/downloads/<videoId>/ with metadata.json + video file
- Support quality selection (1080p, 720p, 480p, audio-only)
- Background download with progress tracking

## Dashboard
Path: /youtube
Layout:
- Search bar at top (channels/videos)
- Channel view: channel header + scrollable video grid with thumbnails, duration, views
- Video detail: title, description, quality picker, download button
- Downloads library: grid of downloaded videos, click to play
- Offline player: video player with keyboard shortcuts (space, fullscreen, seek, volume)

Key interactions:
- Search → pick channel → browse videos → download → play offline
